# JDBC Employee Management System

## Description
This is a Java JDBC-based project that performs CRUD operations on employee data
using a MySQL database. The application allows adding, viewing, updating, and
deleting employee records.

## Features
- Add Employee
- View Employees
- Update Employee
- Delete Employee

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

## How to Run
1. Clone the repository
2. Open the project in Eclipse
3. Configure database connection in the code
4. Run the `Home.java` file
